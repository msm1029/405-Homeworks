    Vocab Definitions

    • HTTP
      • Definition:The Way Computers talk back and forth to each other on the internet
      • Official Docs
    • HTML
      • Definition: A language that determines webpage structure
      • Official Docs
    • Element
      • Definition: A line of data in a webpage
      • Official Docs
    • Tag
      • Definition: Used to open and close an element
      • Official Docs
    • Block
      • Definition: An HTML element that is placed on a new line
      • Official Docs
    • Doctype
      • Definition: A line at the beginning of an HTML Document that ensures the document is rendered correctly
      • Official Docs
    • Head
      • Definition: A line that contains metadata
      • Official Docs
    • Attribute
      • Definition: An extention on a tag that can change its behavior
      • Official Docs
    • Boolean
      • Definition: A line of data that can only have two values. True and False.
      • Official Docs
    • DOM
      • Definition: An API that represents and interacts with an HTML document
      • Official Docs
    • CSS
      • Definition: A language that determines how a webpage looks
      • Official Docs
    • Property (CSS)
      • Definition: A characteristic that you can apply to a webpage
      • Official Docs
    • Selector (CSS)
      • Definition: Something that indicates which elements to apply a rule to.
      • Official Docs
